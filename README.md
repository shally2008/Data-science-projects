## 5. Career Decision Support Tool (Excel)

### Project Description

A simple decision-support project developed in Microsoft Excel to assist users in selecting the most suitable career path based on multiple evaluation criteria.

Different career options are scored using weighted factors such as:

- Salary potential
- Job market demand
- Personal interest
- Educational requirements
- Enjoyment level

Each factor contributes to an overall score, allowing the user to compare different career paths objectively and make a well-informed decision.

### Features

- Weighted scoring system
- Career comparison
- Decision support analysis
- Automatic total score calculation

### Skills Demonstrated

- Excel formulas
- Weighted decision analysis
- Analytical thinking
- Decision support systems
